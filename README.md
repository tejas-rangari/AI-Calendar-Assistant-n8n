# AI Calendar Agent using n8n

This project demonstrates an AI-powered calendar assistant workflow built using n8n that automatically schedules events in Google Calendar.

The assistant has been enhanced using system messages to customize its personality and behavior through prompt engineering.

## Features

- AI-powered calendar event scheduling
- Google Calendar integration
- Workflow automation using n8n
- Custom assistant personality using system messages
- Prompt engineering for response control

## Tech Stack

- n8n
- Google Calendar API
- AI Agent Workflow
- Prompt Engineering

## Workflow Architecture

User Input → AI Agent → Google Calendar Event Creation

## Purpose of Project

This project demonstrates how AI agents can be integrated with automation workflows and external APIs like Google Calendar to build intelligent scheduling assistants.
